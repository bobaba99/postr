# Postr v1 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build production Postr — academic poster web app with anonymous-first auth, autosave, custom presets, asset uploads, reusable libraries, shareable links, and AI-powered poster scan — porting the prototype into a modular full-stack architecture.

**Architecture:** Vite + React + TS SPA on Vercel talks directly to Supabase (auth, Postgres, Storage) for 80% of operations. A small Express API on Render handles only the Claude vision scan endpoint to keep the Anthropic key off the browser. Anonymous Supabase sessions are created on first visit, an `Untitled Poster` is auto-spawned by a Postgres trigger, and the editor opens straight into it — zero signup, zero dialogs.

**Tech Stack:** Vite, React 18, TypeScript, Tailwind CSS, Zustand, React Query, React Router v6, `@supabase/supabase-js`, Express, `@anthropic-ai/sdk` (Claude Sonnet 4.6 with vision), Vitest, Playwright.

**Environment notes (v1):**

- **Supabase:** local-only via `supabase start` for the entire build. Production Supabase project is provisioned during Phase 11.
- **Domain:** `postr.sh` (already purchased). Used in Phase 11 Vercel deploy.
- **LLM:** Anthropic Claude Sonnet 4.6 via `@anthropic-ai/sdk` with image content blocks and tool-use structured output.

## Cross-cutting requirements

- **TDD:** every helper, hook, and API route has a failing test before implementation.
- **Coverage:** 80% minimum across `apps/web/src` and `apps/api/src` (Vitest `--coverage`).
- **Lint:** ESLint + Prettier on commit. Hooks already in `~/.claude/settings.json`.
- **Commits:** one logical commit per task. Conventional commit format (`feat`, `fix`, `chore`, `refactor`, `test`, `docs`).
- **Secrets:** never in repo. `.env.example` only.
- **A11y:** all interactive elements keyboard-reachable; focus styles on selection.
- **Friction audit (after every phase):** does this add a click that didn't exist before? If yes, justify or remove.
